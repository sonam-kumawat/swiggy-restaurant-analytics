# Swiggy-Restaurant-Analytics

## Overview
This project focuses on analyzing Swiggy restaurant data using Python, SQL, Excel, and Jupyter Notebook. The dataset was cleaned, transformed, and normalized to extract meaningful insights related to restaurant performance, pricing trends, customer ratings, delivery time, and popular cuisines.

---

## Project Objectives
- Clean and preprocess raw restaurant data
- Handle inconsistent and missing values
- Normalize multi-value cuisine categories
- Perform SQL-based analytical queries
- Extract business insights from restaurant datasets

---

## Dataset Information
The dataset was collected from Kaggle and contains restaurant-related information such as:

- Restaurant Name
- City and Area
- Price
- Average Ratings
- Total Ratings
- Delivery Time
- Food Categories

---

## Data Preprocessing
Performed data cleaning and preprocessing using Python and Jupyter Notebook:

- Removed duplicate records
- Handled missing and inconsistent values
- Cleaned restaurant and cuisine data
- Split multi-value food categories into separate structured records
- Created normalized CSV tables for efficient SQL analysis

---
